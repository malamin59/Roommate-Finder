# 🏠 Roommate Finder App

A web platform to help users find compatible roommates based on budget, lifestyle, and preferences.

## 🚀 Live Preview
👉 [Live Link](https://roommate-app.web.app)

## 🛠️ Tech Stack
- React.js
- Tailwind CSS
- Firebase
- SweetAlert2
- Framer Motion
